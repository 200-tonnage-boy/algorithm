## 选择排序
  遍历数组，将每次遍历的最小或者最大值放到最前面，因此复杂度为N^2；可以依次遍历，然后调换顺序；

~~~JavaScript
function selectSort(arr) {
  let min = [0, arr[0]];
  for (let i = 0; i < arr.length - 1; i++) {
    min = [i, arr[i]];
    for (let j = i + 1; j < arr.length; j++) {
      if (arr[j] <= min[1]) {
        min = [j, arr[j]];
      }
    }
    if (min[0] !== i) {
      [arr[i], arr[min[0]]] = [arr[min[0]], arr[i]];
    }
  }
  return arr;
}
~~~
